How it Works

  1. Login with JWT-secured account.
  2. Load project via File dropdown (file/folder/upload).
  3. Run DEBUG to lint, SUGGESTION to invoke CodeT5, PREVIEW for web files.
  4. Apply or reject fixes; stream inference logs in console.

Stack Preview

  • Frontend: HTML, CSS, Vanilla JS, Monaco, diff-match-patch.
  • Backend: Flask + Socket.IO for lint + model orchestration.
  • DB: Firebase Firestore for user projects & suggestion metadata.
  • Model: CodeT5-small fine-tune for lint fixes.
